## 题目

给定一个数组 A[0,1,...,n-1],请构建一个数组 B[0,1,...,n-1],其中 B 中的元素 B[i]=A[0]_A[1]_...*A[i-1]*A[i+1]*...*A[n-1]。不能使用除法。（注意：规定 B[0] = A[1] _ A[2] _ ... _ A[n-1]，B[n-1] = A[0] _ A[1] _ ... _ A[n-2];）
对于 A 长度为 1 的情况，B 无意义，故而无法构建，因此该情况不会存在。

## 思路
